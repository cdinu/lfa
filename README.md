## Install

```sh
$ npm install --save lfanew
```


## Usage

```js
var lfanew = require('lfanew');

lfanew('Rainbow');
```

```sh
$ npm install --global lfanew
$ lfanew --help
```

```sh
# creates a browser.js
$ npm run browser
```
